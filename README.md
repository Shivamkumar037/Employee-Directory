# Employee Directory

Employee Directory is a simple Java-based project developed to manage employee records in an organized way. This project helps in storing, viewing, and managing employee details efficiently. It is useful for beginners who want to understand how basic employee management systems work using Java.

## Features

- Add employee details
- View employee records
- Manage employee information
- Simple and easy-to-understand structure
- Console-based Java application
- Beginner-friendly project

## Technologies Used

- Java
- Object-Oriented Programming
- Console Input/Output

## Project Overview

This project is designed to maintain employee information in a structured format. It demonstrates how employee data can be handled in a Java application using classes, objects, and basic programming concepts. The system makes it easier to organize and access employee details.

## How It Works

1. The system allows users to enter employee details.
2. Employee information is stored in the system.
3. Users can view and manage the stored employee records.
4. The application helps in maintaining employee data in a simple format.
